# E-commerce Product Photo Automation Tool

Python script for batch processing product images:

- Automatic background removal (using rembg AI)
- Resize to standard dimensions with white background
- Add logo watermark in corner

Perfect for Amazon, Shopify, Etsy sellers processing hundreds of photos.
